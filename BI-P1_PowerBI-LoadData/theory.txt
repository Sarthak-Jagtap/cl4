1. AIM

To import data from different sources such as Excel, CSV, and OData Feed into Power BI Desktop and load the data into the target system for analysis and visualization.

--------------------------------------------------------------------
3. THEORY
--------------------------------------------------------------------

Business organizations store data in different formats and systems such as Excel files, CSV files, SQL databases, APIs, and cloud platforms. Before analysis can be performed, this data must be imported into a centralized system.

Power BI provides the ability to connect to multiple data sources and load them into a single environment for analysis and reporting.

This process generally includes:

1. Data Extraction
   Retrieving data from various sources.

2. Data Transformation
   Cleaning and modifying data.

3. Data Loading
   Loading processed data into Power BI model.

The imported data can then be visualized using charts, graphs, dashboards, and reports.

The following sources are used:
1. Excel File

This practical demonstrates how Power BI can integrate heterogeneous data sources into a unified reporting system.

5. STEPS TO EXECUTE PRACTICAL
--------------------------------------------------------------------

=================================================
A) IMPORTING EXCEL DATA INTO POWER BI
=================================================

Step 1:
Open Power BI Desktop.

Step 2:
Click on:
Home → Get Data

Step 3:
Select:
Excel Workbook

Step 4:
Click Connect.

Step 5:
Browse and select:
products.xlsx

Step 6:
Click Open.

Step 7:
Navigator window appears.

Step 8:
Select the required sheet/table.

Step 9:
Click:
Transform Data

Step 10:
Power Query Editor opens.

Step 11:
Verify imported columns and data.

Step 12:
Click:
Close & Apply

Step 13:
Data is loaded into Power BI model.

-------------------------------------------------

OUTPUT:
- Navigator Window
- Imported Excel Table
- Query Editor
- Loaded Data

6. EXPLANATION OF EACH IMPORTANT STEP
--------------------------------------------------------------------

-------------------------------------------------
1. Get Data
-------------------------------------------------

Purpose:
Used to connect Power BI with external data sources.

Reason:
Without importing data, analysis cannot be performed.

-------------------------------------------------
2. Transform Data
-------------------------------------------------

Purpose:
Opens Power Query Editor.

Reason:
Allows cleaning and preprocessing before loading data.

-------------------------------------------------
3. Remove Columns
-------------------------------------------------

Purpose:
Deletes unnecessary columns.

Reason:
Reduces memory usage and improves report clarity.

-------------------------------------------------
4. Change Data Types
-------------------------------------------------

Purpose:
Ensures columns contain correct data format.

Example:
- Date column → Date Type
- Quantity → Integer

Reason:
Wrong data types can produce incorrect calculations.

-------------------------------------------------
5. Close & Apply
-------------------------------------------------

Purpose:
Loads transformed data into Power BI.

Reason:
Finalizes ETL process.

-------------------------------------------------
6. Visualization
-------------------------------------------------

Purpose:
Represents data graphically.

Reason:
Helps identify trends and patterns.

7. ETL PROCESS THEORY
--------------------------------------------------------------------

=================================================
1. EXTRACT
=================================================

Definition:
Process of retrieving data from sources.

Examples:
- Excel
- CSV
- Database
- API
- OData Feed

Purpose:
Collect raw data.

=================================================
2. TRANSFORM
=================================================

Definition:
Process of cleaning and modifying data.

Operations:
- Removing duplicates
- Filtering
- Renaming columns
- Changing data types
- Handling null values

Purpose:
Prepare data for analysis.

=================================================
3. LOAD
=================================================

Definition:
Process of loading processed data into target system.

Purpose:
Store transformed data for reporting and analysis.

11. POSSIBLE VIVA QUESTIONS WITH ANSWERS
--------------------------------------------------------------------

Q1. What is Power BI?

Answer:
Power BI is a business analytics and visualization tool developed by Microsoft. It is used to connect, transform, analyze, and visualize data from multiple sources.

-------------------------------------------------

Q2. What is ETL?

Answer:
ETL stands for:
- Extract
- Transform
- Load

It is a process used to collect, clean, and store data for analysis.

-------------------------------------------------

Q3. What is OData Feed?

Answer:
OData (Open Data Protocol) is a standard protocol used to access and share data over the internet.

-------------------------------------------------

Q4. What is Query Editor?

Answer:
Query Editor is a Power BI tool used for transforming and cleaning data before loading it into the model.

-------------------------------------------------

Q5. Why is data transformation important?

Answer:
Raw data may contain errors, duplicates, or unwanted columns. Transformation improves data quality and accuracy.

-------------------------------------------------

Q6. What is the purpose of visualization?

Answer:
Visualization helps represent data graphically so users can understand patterns, trends, and insights easily.

-------------------------------------------------

Q7. Difference between Excel and CSV?

Answer:
Excel:
- Supports multiple sheets
- Supports formatting and formulas

CSV:
- Plain text format
- Stores comma-separated values

-------------------------------------------------

Q8. What is the use of Remove Other Columns?

Answer:
It keeps only selected columns and removes remaining unnecessary columns.

-------------------------------------------------

Q9. What happens when Close & Apply is clicked?

Answer:
Power BI applies all transformations and loads data into the model.

-------------------------------------------------

Q10. What are the benefits of Power BI?

Answer:
- Interactive dashboards
- Multiple data source support
- Fast analysis
- Easy reporting
- Real-time insights
