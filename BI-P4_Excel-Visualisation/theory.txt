3. THEORY
--------------------------------------------------------------------

Data Visualization is the graphical representation of data using charts, graphs, and tables.

Visualization helps:
1. Understand trends
2. Compare values
3. Identify patterns
4. Detect outliers
5. Improve decision making

Excel provides various visualization tools such as:
- Bar Charts
- Column Charts
- Line Charts
- Pie Charts
- Scatter Plots
- Pivot Tables

These charts convert raw numerical data into meaningful visual information.
